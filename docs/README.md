# 个人情报官 - 项目索引

## 项目简介

**个人情报官**是基于RSSHub扩展的RSS订阅管理平台，提供用户系统、订阅管理、后台管理等功能。

## 开发工作流

### 新人上手步骤

1. **阅读项目索引**（本文件）
2. **阅读需求文档** - 了解项目做什么
3. **阅读技术方案** - 了解系统架构
4. **阅读开发规范** - 了解代码标准
5. **开始开发** - 从README.md启动项目

### 开发流程

```
需求确认 → 技术方案设计 → 审批 → 开发 → 测试 → 部署
```

### 开发日志记录

每次开发会议或重要修改后，记录到 `docs/开发日志/YYYY-MM-DD.md`
