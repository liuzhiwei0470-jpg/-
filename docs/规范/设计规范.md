# 设计规范

## 一、UI规范

### 1.1 色彩系统

#### 主色
| 用途 | 色值 | 说明 |
|------|------|------|
| Primary | #4F46E5 | 主按钮、链接、强调 |
| Primary Hover | #4338CA | 主色悬停状态 |
| Primary Light | #EEF2FF | 浅色背景 |

#### 中性色
| 用途 | 色值 |
|------|------|
| 文字主色 | #1F2937 |
| 文字次色 | #6B7280 |
| 文字弱色 | #9CA3AF |
| 边框色 | #E5E7EB |
| 背景色 | #F9FAFB |
| 白色 | #FFFFFF |

#### 语义色
| 用途 | 色值 |
|------|------|
| 成功 | #10B981 |
| 警告 | #F59E0B |
| 错误 | #EF4444 |
| 信息 | #3B82F6 |

### 1.2 字体系统

| 用途 | 字体 | 大小 | 行高 |
|------|------|------|------|
| 标题1 | Inter | 24px | 1.2 |
| 标题2 | Inter | 20px | 1.3 |
| 标题3 | Inter | 16px | 1.4 |
| 正文 | Inter | 14px | 1.5 |
| 小字 | Inter | 12px | 1.4 |

### 1.3 间距系统

使用4px基准网格：
- `xs`: 4px
- `sm`: 8px
- `md`: 16px
- `lg`: 24px
- `xl`: 32px
- `2xl`: 48px

### 1.4 圆角

| 组件 | 圆角 |
|------|------|
| 按钮 | 6px |
| 输入框 | 6px |
| 卡片 | 8px |
| 弹窗 | 12px |

## 三、交互规范

### 3.1 按钮

| 类型 | 样式 | 用途 |
|------|------|------|
| Primary | 背景Primary色，白色文字 | 主要操作 |
| Secondary | 白色背景，Primary边框 | 次要操作 |
| Ghost | 透明背景，Primary文字 | 辅助操作 |
| Danger | 错误色背景 | 危险操作 |

### 3.2 表单

- 必填字段加 `*` 标记
- 错误提示在字段下方
- 成功反馈使用 Toast

### 3.3 列表

- 空状态显示友好提示
- 加载状态显示 Skeleton
- 分页使用数字分页器

### 3.4 导航

- 当前页面高亮显示
- 支持键盘导航
- 移动端使用汉堡菜单

## 四、响应式断点

| 设备 | 宽度 | 布局 |
|------|------|------|
| 移动端 | < 768px | 单列 |
| 平板 | 768px - 1024px | 双列 |
| 桌面 | > 1024px | 多列 |

## 五、图标

- 使用 Lucide Icons（SVG）
- 统一尺寸：16px / 20px / 24px
- 颜色跟随文字颜色

## 六、动画

|------|------|
